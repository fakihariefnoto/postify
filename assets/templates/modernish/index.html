Kenapa blog?

Selamat datang di halaman pribadi saya. jadi kenapa halaman ini berupa blog?

Karena blogging adalah salah satu cara kita untuk mengekspresikan pikiran dan ide secara bebas dan tertulis agar dikemudian hari kita bisa mengingatnya kembali dengan membacanya lagi.

Sebuah ilmu yg kita dapat mungkin akan cepat hilang bila dipendam saja, kalo bisa di-sharing, kenapa tidak? pada blog ini saya bakal memfokuskan ke bidang yg saya tekuni saat ini yaitu software development. Semoga bermanfaat.

Beberapa tema yg saya sukai:

  • Software Development
  • Programming
  • Scripting
  • Smart System
  • System Design
  • Protocol Communication
